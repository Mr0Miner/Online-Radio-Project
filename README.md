# پروژه رادیۆی آنلاین

ئەم پرۆژەیە رادیۆیەکی آنلاینە کە بەرەوپێشکاران ئەم شێوەیە بەرز دەکات بۆ گۆرانییەکان.

## تایبەتمەندییەکان
- پخش گۆرانییەکان بە شێوەیەکی بەرز
- لیستی گۆرانییەکان دڵخواز
- توانای لایک کردنی گۆرانییەکان
- پەیوەندیدانی بەرز و بەرز

## نووسەر
